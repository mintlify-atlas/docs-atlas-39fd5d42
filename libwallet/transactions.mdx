## Overview

Muun's transaction system uses Partially Signed Transactions (PST) to coordinate signing between user and server. This ensures both parties validate the transaction before signing.

## PartiallySignedTransaction

### Structure

```go
type PartiallySignedTransaction struct {
    tx     *wire.MsgTx  // Bitcoin transaction
    inputs []Input       // Input metadata
    nonces *MusigNonces  // For MuSig2 signing
}
```

### Creation

<CodeGroup>

```go Go
func NewPartiallySignedTransaction(
    inputs *InputList,
    rawTx []byte,
    userNonces *MusigNonces,
) (*PartiallySignedTransaction, error) {
    tx := wire.NewMsgTx(0)
    err := tx.Deserialize(bytes.NewReader(rawTx))
    if err != nil {
        return nil, fmt.Errorf("failed to decode tx: %w", err)
    }
    
    return &PartiallySignedTransaction{
        tx:     tx,
        inputs: inputs.Inputs(),
        nonces: userNonces,
    }, nil
}
```

## Transaction Signing

### Sign with User Key

Sign transaction inputs with user's private key:

```go
func (p *PartiallySignedTransaction) Sign(
    userKey *HDPrivateKey,
    muunKey *HDPublicKey,
) (*Transaction, error) {
    coins, err := p.coins(userKey.Network)
    if err != nil {
        return nil, fmt.Errorf("could not convert input data to coin: %w", err)
    }
    
    for i, coin := range coins {
        err = coin.SignInput(i, p.tx, userKey, muunKey)
        if err != nil {
            return nil, fmt.Errorf("failed to sign input: %w", err)
        }
    }
    
    return newTransaction(p.tx)
}
```

### Full Signing (Recovery)

Sign with both keys for recovery:

```go
func (p *PartiallySignedTransaction) FullySign(
    userKey, muunKey *HDPrivateKey,
) (*Transaction, error) {
    coins, err := p.coins(userKey.Network)
    if err != nil {
        return nil, fmt.Errorf("could not convert input data to coin: %w", err)
    }
    
    for i, coin := range coins {
        err = coin.FullySignInput(i, p.tx, userKey, muunKey)
        if err != nil {
            return nil, fmt.Errorf("failed to sign input: %w", err)
        }
    }
    
    return newTransaction(p.tx)
}
```

## Transaction Validation

### SigningExpectations

Define what you expect in the transaction:

```go
type SigningExpectations struct {
    destination string      // Recipient address
    amount      int64       // Amount to send
    change      MuunAddress // Change address
    fee         int64       // Expected fee
    alternative bool        // Alternative TX flag
}
```

### Verification

Validate transaction before signing:

<Tabs>
  <Tab title="Go">
    ```go
    expectations := &SigningExpectations{
        destination: "bc1qxy2kgdygjrsqtzq2n0yrf2493p83kkfjhx0wlh",
        amount:      100000, // 0.001 BTC
        change:      changeAddress,
        fee:         1000,
        alternative: false,
    }
    
    err := pst.Verify(expectations, userPubKey, muunPubKey)
    if err != nil {
        return fmt.Errorf("transaction verification failed: %w", err)
    }
    ```
  </Tab>
  
  <Tab title="Swift">
    ```swift
    let expectations = SigningExpectations(
        destination: "bc1qxy2kgdygjrsqtzq2n0yrf2493p83kkfjhx0wlh",
        amount: 100000,
        change: changeAddress,
        fee: 1000,
        alternative: false
    )
    
    do {
        try pst.verify(
            expectations,
            userPublicKey: userPubKey,
            muunPublickKey: muunPubKey

        print("Verification failed: \(error)")
    ```

  <Accordion title="Output Count Validation">

  <Accordion title="Destination Validation">

    ```go
    if err != nil {
        return err

    for _, output := range p.tx.TxOut {
        if bytes.Equal(output.PkScript, toScript) {
            toOutput = output
            break
        }
    }
    
    if toOutput == nil {
        return errors.New("destination output is not present")
    }
    ```

  </Accordion>

## Coin Types

Different address versions require different signing logic:

### Coin Interface

```go
type coin interface {
    SignInput(index int, tx *wire.MsgTx,
        userKey *HDPrivateKey, muunKey *HDPublicKey) error
    FullySignInput(index int, tx *wire.MsgTx,
        userKey, muunKey *HDPrivateKey) error
}
```

### Creating Coins

```go
func createCoin(
    index int,
    input Input,
    network *Network,
    sigHashes *txscriptw.TaprootSigHashes,
    userNonces *MusigNonces,
) (coin, error) {
    version := input.Address().Version()
    
    switch version {
    case addresses.V1:
        return &coinV1{...}, nil
    case addresses.V2:
        return &coinV2{...}, nil
    case addresses.V3:
        return &coinV3{...}, nil
    case addresses.V4:
        return &coinV4{...}, nil
    case addresses.V5:
        return &coinV5{...}, nil
    case addresses.V6:
        return &coinV6{...}, nil
    case addresses.SubmarineSwapV1:
        return &coinSubmarineSwapV1{...}, nil
    case addresses.SubmarineSwapV2:
        return &coinSubmarineSwapV2{...}, nil
    default:
        return nil, fmt.Errorf("unsupported version %v", version)
    }
}
```

### V5 Coin Example

```go
type coinV5 struct {
    Network        *chaincfg.Params
    OutPoint       wire.OutPoint
    KeyPath        string
    Amount         btcutil.Amount
    UserSessionId  [32]byte
    MuunPubNonce   [66]byte
    MuunPartialSig [32]byte
    SigHashes      *txscriptw.TaprootSigHashes
}

func (c *coinV5) SignInput(
    index int,
    tx *wire.MsgTx,
    userKey *HDPrivateKey,
    muunKey *HDPublicKey,
) error {
    // Derive keys to path
    derivedUserKey, err := userKey.DeriveTo(c.KeyPath)
    derivedMuunKey, err := muunKey.DeriveTo(c.KeyPath)
    
    // Calculate sighash
    sigHash, err := txscriptw.CalcTaprootSigHash(
        tx, c.SigHashes, index, txscript.SigHashAll,
    )
    
    // Sign with MuSig2
    return c.signSecondWith(
        index, tx,
        userEcPriv, muunEcPub,
        c.UserSessionId, sigHash,
    )
}
```

## Transaction Result

### Transaction Type

```go
type Transaction struct {
    Hash  string
    Bytes []byte
}
```

### Serialization

```go
func newTransaction(tx *wire.MsgTx) (*Transaction, error) {
    var buf bytes.Buffer
    err := tx.Serialize(&buf)
    if err != nil {
        return nil, fmt.Errorf("failed to encode tx: %w", err)
    }
    
    return &Transaction{
        Hash:  tx.TxHash().String(),
        Bytes: buf.Bytes(),
    }, nil
}
```

## Dust Threshold

Muun enforces a dust threshold to prevent uneconomical outputs:

```go
const dustThreshold = 546 // satoshis
```

Outputs smaller than this are considered "dust" and may be rejected.

## Alternative Transactions

Alternative transactions allow flexible fee adjustment:

```go
func (e *SigningExpectations) ForAlternativeTransaction() *SigningExpectations {
    return &SigningExpectations{
        e.destination,
        e.amount,
        e.change,
        e.fee,
        true, // alternative flag
    }
}
```

Alternative TXs:
- May omit destination output if amount too small
- Move reduced amount to fee
- Help with fee bumping

## Best Practices

<AccordionGroup>
  <Accordion title="Always Verify Before Signing">
    Never sign a transaction without verifying it matches expectations:
    ```swift
    do {
        try pst.verify(expectations, userPubKey, muunPubKey)
        let signedTx = try pst.sign(userPrivKey, muunPubKey)
        broadcast(signedTx)
    } catch {
        // Handle verification failure
    }
    ```
  </Accordion>
  
  <Accordion title="Generate Fresh Nonces">
    Always use cryptographically random nonces for MuSig2:
    ```swift
    import CryptoKit
    
    let sessionId = Data(SymmetricKey(size: .bits256))
    try nonces.add(sessionId)
    ```
  </Accordion>
  
  <Accordion title="Check Network Match">
    Ensure all addresses match the network:
    ```swift
    guard address.network == .mainnet else {
        throw TransactionError.networkMismatch
    }
    ```
  </Accordion>
  
  <Accordion title="Validate Change Derivation">
    Always verify change addresses are properly derived:

    ```go
    if changeOutput != nil {
    ```
