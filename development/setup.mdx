This guide walks you through setting up your development environment for the Muun Wallet iOS application.

## Prerequisites

Before you begin, ensure you have the following tools installed:

<Steps>
  <Step title="Install Xcode">
    Download and install Xcode from the Mac App Store. Make sure you have Xcode command line tools installed:
    
    ```bash
    xcode-select --install
    ```
    
    Verify the command line tools location in **Xcode > Preferences > Locations > Command Line tools**
  </Step>

  <Step title="Install Golang">
    The libwallet module is written in Go. Install Go from [golang.org](https://golang.org/):
    
    ```bash
    brew install go
    ```
    
    Set up your `GOPATH` environment variable:
    
    ```bash
    export GOPATH=$HOME/go
    export PATH=$PATH:$GOPATH/bin
    ```
  </Step>

  <Step title="Install Docker">
    Docker is required to build the Rust components in libwallet. Install from [docs.docker.com](https://docs.docker.com/engine/install/):
    
    ```bash
    brew install --cask docker
    ```
  </Step>

  <Step title="Install CocoaPods">
    CocoaPods manages iOS dependencies. Install it using:
    
    ```bash
    sudo gem install cocoapods
    ```
    
    Or use bundler (recommended):
    
    ```bash
    bundle install
    ```
  </Step>
</Steps>

## Repository Setup

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/muun/falcon.git
    cd falcon
    ```
  </Step>

  <Step title="Build libwallet Rust components">
    The libwallet module includes Rust code that needs to be compiled for iOS:
    
    ```bash
    cd libwallet/librs
    ./makelibs.sh
    ```
    
    This script:
    - Builds a Docker image with necessary Rust toolchains
    - Compiles the verifier for iOS targets (aarch64-apple-ios, x86_64-apple-ios, aarch64-apple-ios-sim)
    - Creates an `.xcframework` bundle for Xcode
    
    <Note>
      The first run takes 10-15 minutes as it compiles for multiple architectures. Subsequent builds use caching.
    </Note>
  </Step>

  <Step title="Install gomobile">
    Bootstrap gomobile to bind the Go libwallet code:
    
    ```bash
    cd ../..
    ./tools/bootstrap-gomobile.sh
    ```
    
    This installs `gomobile` and `gobind` to your `GOPATH/bin`.
  </Step>

  <Step title="Install pod dependencies">
    Navigate to the app directory and install CocoaPods dependencies:
    
    ```bash
    cd falcon/app
    pod install
    ```
    
    This command:
    - Installs all dependencies defined in `Podfile`
    - Stores your `GOPATH` in `.gopath` file for build scripts
    - Runs `tools/libwallet-ios.sh` to build the Go libwallet framework
    - Generates the pod dependency registry
  </Step>

  <Step title="Open the workspace">
    Open the workspace file (not the `.xcodeproj`):
    
    ```bash
    open falcon.xcworkspace
    ```
    
    <Note>
      Always use `falcon.xcworkspace` when working with CocoaPods projects.
    </Note>
  </Step>
</Steps>

## Environment Variables

The build system relies on the `GOPATH` environment variable. The `pre_install` hook in the Podfile stores this value:

```ruby Podfile
pre_install do |installer|
  # Store GOPATH for build scripts
  File.open('.gopath', 'w') {|f| f.write(ENV["GOPATH"]) }
  
  # Build libwallet framework
  `../../tools/libwallet-ios.sh`
end
```

## Verification

Verify your setup by building the project:

1. Select the **Muun** scheme in Xcode
2. Choose a simulator or device
3. Press **Cmd+B** to build
4. Press **Cmd+R** to run

If the build succeeds, you're ready to develop!

## Troubleshooting

<Accordion title="gomobile not found">
  Ensure `$GOPATH/bin` is in your PATH:
  
  ```bash
  echo $PATH | grep $GOPATH/bin
  ```
  
  If not found, add to your shell profile:
  
  ```bash
  export PATH=$PATH:$GOPATH/bin
  ```
</Accordion>

<Accordion title="Docker not running">
  The librs build requires Docker. Start Docker Desktop or the Docker daemon:
  
  ```bash
  open -a Docker
  ```
</Accordion>

<Accordion title="Pod install fails">
  Try cleaning and reinstalling:
  
  ```bash
  pod deintegrate
  pod cache clean --all
  pod install
  ```
</Accordion>

<Accordion title="Libwallet build errors">
  The libwallet framework is built automatically during `pod install`. If you see errors:
  
  1. Check that `GOPATH` is set correctly
  2. Manually run the build script:
     ```bash
     ./tools/libwallet-ios.sh
     ```
  3. Check that gomobile is installed:
     ```bash
     which gomobile
     ```
</Accordion>
